# The Registry

a browsable community of artifacts typed by contracts and graded by conformance, self-hosting the ecosystem's own evolution machinery, expressible from inside the app by anyone, this app's own client entry included, graded by the identical pair, with no privileged placement

Domain: artifacts of the ecosystem itself: extensions, skins, components, clients, and communities

This is an EpiStack community kernel, self-contained (vendor/kernel/ is its own vendored copy). Its snapshot is at snapshot/the-registry.snapshot.json; its community card is community-card.json. A pull request against this repository carrying a contribution bundle re-runs the gate (build/check.mjs) before merge.
